# Login Background Image

## Instructions

1. Save your futuristic red/black background image to this directory
2. Name it: `login-background.jpg`
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Image Specifications

- **Recommended size**: 1920x1080 or larger
- **Format**: JPG, PNG, or WebP
- **File name**: `login-background.jpg`

The login page will automatically use this image as the background. If the image is not found, a dark red/black gradient fallback will be displayed.
