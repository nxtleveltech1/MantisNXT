# Supplier Test Cases

Fixture data for supplier pricelist validation and import debugging. Each test case pairs a markdown description with CSV (and sometimes XLSX) sample data for a given supplier and date.

- **Purpose:** Reproducible test data for NXT-SPP uploads, extraction, and selection flows.
- **Naming:** `{supplier}-test-case-{YYYY-MM-DD}[-vN].{md,csv,xlsx}`

Do not commit large production datasets here—keep only minimal samples needed for validation.
